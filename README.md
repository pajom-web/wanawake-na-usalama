# Real-Time Web-Based Gender-Sensitive Safety Mobility System for Women

An urban safety and mobility web platform designed to enhance safety for women by providing location based tracking, safety reporting, and data driven mobility insights.


## Features

Real Time Location Tracking: Monitor safe transit routes and user locations.
Safety Incident Reporting: Log and map user reported safety incidents in real time.
Interactive Dashboard: Visual analytics displaying spatial safety scores and mobility trends.
Lightweight Architecture: Built for high efficiency, fast response times, and simple local deployment without heavy external service dependencies.



##  Tech Stack

Frontend: [Flutter]Web
Backend:[Python-Django]
Database: SQLite (Default localized database)
